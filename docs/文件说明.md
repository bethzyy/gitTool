## 🎯 启动方式

### 桌面应用

**双击运行**（推荐）：


```bash
python git_gui_app.py
```

## 📁 运行时生成的文件

**`logs/`** - 日志目录
- `app-YYYY-MM-DD.log` - 应用运行日志
- 记录所有操作和错误信息
- 方便问题追溯

**`__pycache__/`** - Python 缓存
- 编译的字节码
- Python 自动生成和管理

## ✨ 使用流程

### 第一次使用

1. 双击 `启动应用.bat`
2. 如提示缺少 Python 或 Git，先安装对应软件
3. 配置 Git 用户信息（如需要）
4. 开始使用

### 日常使用

1. 双击 `启动应用.bat`
2. 填写仓库地址、提交信息、代码路径
3. 点击"提交到 GitHub"
4. 等待完成

### 开发调试

1. 运行 `scripts/test_security_scan.py` 测试扫描
2. 查看 `logs/` 目录中的日志
3. 参考 `docs/CLAUDE.md` 修改代码

## 🔍 故障排查

### 应用无法启动

1. 检查 Python：`python --version`
2. 检查 Git：`git --version`
3. 查看 `logs/` 中的日志文件

### 功能异常

1. 查看应用内日志输出区域
2. 查看 `logs/` 详细日志
3. 参考 `docs/使用说明.md` 常见问题

### 安全扫描误报

1. 运行 `scripts/test_security_scan.py` 查看详情
2. 检查是否使用了示例代码
3. 参考 `docs/CLAUDE.md` 调整扫描规则
