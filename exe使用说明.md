# Git GUI 提交工具 - EXE 版本

## 📦 文件说明

**文件名**：`GitGUI提交工具.exe`  
**位置**：`dist/GitGUI提交工具.exe`  
**大小**：约 9.7 MB  
**类型**：Windows 可执行文件（独立运行）  
**版本**：v1.3.0

## 🚀 使用方法

### 方式一：直接运行（推荐）

1. 进入 `dist` 文件夹
2. 双击 `GitGUI提交工具.exe`
3. 应用程序启动，无命令行窗口

### 方式二：创建快捷方式

为了更方便使用，可以创建快捷方式：

1. 右键点击 `GitGUI提交工具.exe`
2. 选择"发送到" → "桌面快捷方式"
3. 以后可以直接从桌面双击运行

### 方式三：固定到任务栏

1. 将快捷方式创建到桌面
2. 右键点击桌面快捷方式
3. 选择"固定到任务栏"
4. 以后可以一键启动

## ✨ 功能特点

- ✅ **独立运行** - 无需安装 Python 环境
- ✅ **无窗口启动** - 只显示图形界面，无命令行窗口
- ✅ **单文件** - 所有依赖打包在一个 exe 中
- ✅ **紧凑界面** - 窗口尺寸550x650，小巧精致
- ✅ **统一输入框** - 所有输入框高度一致，布局整齐
- ✅ **大日志框** - 优化间距，日志显示区域更大
- ✅ **安全扫描** - 默认开启，防止敏感信息泄露
- ✅ **完整日志** - 显示所有 Git 命令

## 📋 前置要求

虽然 exe 文件独立运行，但仍需要：

1. **Git** - [下载地址](https://git-scm.com/downloads)
2. **GitHub SSH 密钥** 或 **Personal Access Token**
3. **配置 Git 用户信息**：
   ```bash
   git config --global user.name "你的名字"
   git config --global user.email "你的邮箱"
   ```

## 🎨 界面优化（v1.3.0）

### 紧凑布局
- 窗口尺寸从 600x700 缩小到 550x650

## 🔧 如何修改 GitHub 用户名

默认使用的用户名是 `bethzyy`，如需修改：

1. 使用文本编辑器打开源码 `git_gui_app.py`
2. 找到第 296 行：
   ```python
   repo_url = f"git@github.com:bethzyy/{repo_name}.git"
   ```
3. 将 `bethzyy` 改为你的 GitHub 用户名
4. 重新运行 `build_exe.bat` 打包

## 📦 重新打包

如果修改了代码，需要重新打包：

### 方式一：使用打包脚本（推荐）

双击运行：`build_exe.bat`

### 方式二：手动打包

```bash
pip install pyinstaller
python -m PyInstaller --onefile --windowed --name "GitGUI提交工具" git_gui_app.py
```

打包完成后，新的 exe 文件在 `dist` 文件夹中。

## 🔍 杀毒软件信任

如果杀毒软件拦截：

1. Windows Defender：
   - 打开"病毒和威胁防护"
   - 点击"保护历史"
   - 找到被拦截的记录
   - 点击"操作" → "允许"

2. 其他杀毒软件：
   - 添加到白名单
   - 或暂时关闭实时保护
