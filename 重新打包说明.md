# 重新打包说明

## 📦 快速打包

### 方式一：使用打包脚本（推荐）

1. **关闭正在运行的 exe**：
   - 关闭所有 `GitGUI提交工具.exe` 窗口
   - 或打开任务管理器结束进程

2. **双击运行**：`build_exe.bat`
   - 会自动清理旧文件
   - 重新打包
   - 显示打包进度和结果

### 方式二：手动打包

1. **删除旧文件**：
   ```bash
   rmdir /s /q build
   del *.spec
   ```

2. **执行打包命令**：
   ```bash
   python -m PyInstaller --onefile --windowed --name "GitGUI提交工具" git_gui_app.py
   ```

3. **查看结果**：
   - 新的 exe 文件在 `dist` 文件夹中
   - 文件名：`GitGUI提交工具.exe`

## 📋 打包参数说明

```bash
--onefile              # 打包成单个 exe 文件
--windowed             # 无控制台窗口（只显示 GUI）
--name "GitGUI提交工具"  # 指定 exe 文件名
git_gui_app.py        # 要打包的 Python 脚本
```

## 📁 打包后的文件结构

```
gitTool/
├── dist/
│   └── GitGUI提交工具.exe    ← 这就是你要运行的文件
├── build/                    ← 临时文件，可删除
└── *.spec                    ← 配置文件，可删除
```

## 💡 提示

- 打包后的 exe 文件约 **9.7 MB**
- 包含了所有依赖，无需安装 Python
- 可以直接分发给其他人使用
- 建议创建桌面快捷方式方便使用

## 🔧 如何修改 GitHub 用户名

默认使用的用户名是 `bethzyy`，如需修改：

1. 使用文本编辑器打开源码 `git_gui_app.py`
2. 找到第 296 行：
   ```python
   repo_url = f"git@github.com:bethzyy/{repo_name}.git"
   ```
3. 将 `bethzyy` 改为你的 GitHub 用户名
4. 重新打包
